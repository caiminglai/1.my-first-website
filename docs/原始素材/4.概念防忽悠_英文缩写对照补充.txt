数学·AI·经济学 概念防忽悠指南
 含英文缩写人话翻译
将数学、计算机、AI、经济学中的术语、英文缩写、营销黑话全部还原为人话。每一个缩写都给出全称和人话翻译，

===== 第2页 =====
前言
本指南涵盖三大领域：数学、计算机/AI、经济学。每一组都给出：术语/缩写 ↔
人话翻译，本质一句话，以及防忽悠提示。英文缩写全部标注全称，并给出中文语境下的等价表达。
核心原则：任何术语，三步之内必还原本质。第一步问"它在描述什么日常现象？"，第二步问"它的数学/物理
/生物本质是什么？"，第三步问"谁在通过这个名字获利？"


===== 第3页 =====
第一部分 数学：抽象结构的多种面孔
函数 (Function)
 ↔
映射 / 输入到输出的对应规则
本质：分析学叫函数，代数学叫映射，几何叫变换
防忽悠：看到映射别慌，就是函数，只是代数人不喜欢说函数因为太分析味
算子 (Operator)
 ↔
作用在函数上的函数
本质：泛函分析里的函数，定义域是函数空间
防忽悠：算子听起来像机器，本质就是函数，只不过输入输出都是函数
泛函 (Functional)
 ↔
函数的函数 / 输出是数的特殊映射
本质：输入是函数、输出是数的映射
防忽悠：泛函=函数的函数，比如曲线的长度、曲面的面积
特征值 / 本征值 (Eigenvalue)
 ↔
矩阵在特定方向上只伸缩不旋转的
倍数
本质：Eigenvalue音译本征，意译特征，同一个东西
防忽悠：Eigen=自己的，value=值，矩阵自己的值=不变方向上的伸缩倍数
张量 (Tensor)
 ↔
多维数组 (ndarray)
本质：AI说张量，程序员说多维数组，NumPy的ndarray就是张量
防忽悠：看到张量就想象一个Excel表格，一维是向量，二维是矩阵，三维以上叫张量
流形 (Manifold)
 ↔
局部像平面的弯曲空间
本质：地球表面是流形：局部是平面（地图），整体是球
防忽悠：广义相对论说时空是弯曲的，数学家说时空是四维流形，一回事
群 (Group)
 ↔
对称性集合
本质：物理说对称性，数学说群；晶体结构的对称性就是点群
防忽悠：群=对称性的数学语言，旋转、翻转、置换都构成群
拓扑 (Topology)
 ↔
橡皮泥几何
本质：研究连续变形下不变的性质，咖啡杯和甜甜圈拓扑相同（一个洞）
防忽悠：拓扑=不关心大小距离，只关心洞有几个、是否连通
希尔伯特空间 (Hilbert Space)
 ↔
无限维的、有角度的、完备的向量
空间

本质：量子力学天天提，本质就是无限维欧氏空间
防忽悠：高中生能理解欧氏空间就能理解它，只是多了无限维和完备性两个条件
概率 / 测度 (Probability / Measure)
 ↔
可能性大小的公理化度量
本质：概率论就是测度论的一个特例（总质量=1）
防忽悠：概率=满足归一化的测度，数学家眼里随机变量就是可测函数
收敛 (Convergence)
 ↔
趋近 / 稳定下来
本质：序列无限接近某个极限
防忽悠：收敛=趋近=稳定下来，数值计算说迭代收敛就是算稳定了
核 (Kernel)
 ↔
零空间 / 解空间
本质：被映射送到零的全体元素
防忽悠：核(Kernel)=零空间(Nullspace)=齐次方程的解空间，三个名字一回事
同构 (Isomorphism)
 ↔
结构相同 / 换个名字一模一样
本质：保持所有结构且可逆的映射
防忽悠：同构=本质上就是同一个东西，只是标签不同
范畴 (Category)
 ↔
对象+箭头 / 抽象关系网络
本质：由对象和它们之间的态射组成的结构
防忽悠：范畴论把一切都抽象成点和箭头，极度抽象但极度统一
傅里叶变换 (Fourier Transform)
 ↔
把波形拆成不同频率的正弦波
本质：信号处理叫频谱分析，光学叫衍射图样
防忽悠：任何波形都能拆成一堆正弦波的和，就像任何颜色都能拆成RGB
第二部分 计算机与AI：黑话与缩写还原
LLM (Large Language Model)
 ↔
大模型 / 超级自动补全
本质：参数量巨大的语言模型，基于概率预测下一个token
防忽悠：LLM=超级自动补全引擎，你输入今天天气，它算概率最大的下一个词是很好
RLHF (Reinforcement Learning from
Human Feedback)
 ↔
让人给AI打分调模型 / 训狗
本质：用人类反馈训练奖励模型，再用强化学习优化


===== 第5页 =====
防忽悠：RLHF=训狗：人给模型打分，模型根据分数调整自己
RAG (Retrieval-Augmented
Generation)
 ↔
开卷考试 / 先查资料再回答
本质：先检索知识库再生成回答，外挂知识库
防忽悠：RAG=开卷考试，模型不会硬编，先去数据库查资料再组织语言
CoT (Chain of Thought)
 ↔
要求AI写草稿纸 / 显式推理
本质：让模型逐步展示推理步骤
防忽悠：CoT=要求模型写草稿纸：第一步…第二步…因此答案…准确率大幅提升
API (Application Programming
Interface)
 ↔
程序之间的插座 / 接口
本质：我给你数据，你按我的规矩来取
防忽悠：API=插座：手机充电口是硬件API，软件API就是数据接口
SaaS (Software as a Service)
 ↔
租软件用 / 不用买
本质：通过互联网提供软件服务
防忽悠：SaaS=租软件：钉钉、腾讯文档，按月付费不用买光盘
GPU (Graphics Processing Unit)
 ↔
显卡 / 并行计算芯片
本质：原本画图，现在用来并行计算矩阵
防忽悠：GPU=显卡，AI训练靠它，因为神经网络全是矩阵乘法，GPU擅长并行
NLP (Natural Language Processing)
 ↔
让电脑理解人话
本质：计算机处理和理解人类语言
防忽悠：NLP=人机对话的基础，语音识别、机器翻译、聊天机器人都属于NLP
CV (Computer Vision)
 ↔
让电脑看懂图片和视频
本质：计算机识别和理解视觉信息
防忽悠：CV=人脸识别、自动驾驶看路、医学影像分析
IoT (Internet of Things)
 ↔
万物互联 / 冰箱灯泡都上网
本质：物理设备通过网络连接和交换数据
防忽悠：IoT=你的空调、手表、体重秤都能连WiFi传数据
神经网络 (Neural Network)
 ↔
复杂函数拟合器 / 万能逼近器
本质：多层非线性函数复合，用数据拟合任意映射


===== 第6页 =====
防忽悠：神经网络听起来像大脑，本质就是一个有很多参数的函数
反向传播 (Backpropagation)
 ↔
链式法则求导 (微积分)
本质：从输出误差反向计算每层参数的梯度
防忽悠：反向传播=复合函数求导的链式法则，高中导数知识，只是层数多用计算机自动算
梯度下降 (Gradient Descent)
 ↔
沿着最陡方向下山
本质：按损失函数的负梯度方向更新参数
防忽悠：梯度下降=看哪边最陡就往反方向走一步，爬山时找最陡的下坡路
损失函数 (Loss Function)
 ↔
误差度量 / 目标函数
本质：衡量模型输出与真实值差距的函数
防忽悠：损失=误差=代价=目标，同一个东西四个名字
正则化 (Regularization)
 ↔
惩罚项 / 先验约束 / 不要想太多
本质：在损失函数里加惩罚防止参数过大
防忽悠：正则化=给模型戴紧箍咒，L2=参数别太大，L1=参数尽量稀疏
过拟合 (Overfitting)
 ↔
死记硬背 / 高方差
本质：模型记住了训练数据的噪声，对新数据表现差
防忽悠：过拟合=死记硬背=只会背题不会变通，把噪声当规律
注意力机制 (Attention)
 ↔
加权求和 (小学平均数升级版)
本质：根据相关性给不同位置分配权重
防忽悠：注意力=加权求和，小学算平均分也是加权求和（权重相等），只是权重由模型自己算
Transformer
 ↔
带加权求和的深度网络
本质：完全基于注意力机制，并行处理序列
防忽悠：Transformer=不用循环、全靠注意力加权求和的网络，GPT和BERT都是它生的
扩散模型 (Diffusion Model)
 ↔
从噪声中逐步修图
本质：通过逐步去噪生成数据
防忽悠：扩散模型=从一张纯雪花屏开始，一点点修成清晰图片，就像雕塑家从大理石中雕出人像
强化学习 (RL / Reinforcement
Learning)
 ↔
试错学习 / 奖惩训练
本质：通过奖励和惩罚学习最优策略


===== 第7页 =====
防忽悠：强化学习=训狗：做对了给零食，做错了打屁股
GAN (Generative Adversarial
Network)
 ↔
造假者与鉴定师的博弈
本质：生成器和判别器互相训练
防忽悠：GAN=造假者（生成器）和鉴定师（判别器）打架，假货越来越真，鉴定师越来越精
联邦学习 (Federated Learning)
 ↔
数据不出本地的协作学习
本质：多个参与方在不共享原始数据的情况下联合训练
防忽悠：联邦学习=各医院数据不出门，只传模型参数，一起训练
知识蒸馏 (Knowledge Distillation)
 ↔
大模型教小模型 / 传功
本质：用大模型的输出训练小模型
防忽悠：蒸馏=老师（大模型）教学生（小模型），把软知识传给小模型
A/B测试 (A/B Testing)
 ↔
对照实验
本质：随机分组比较两种方案
防忽悠：A/B测试=一半用户看A版，一半看B版，看哪个好
CI/CD (Continuous Integration /
Deployment)
 ↔
自动测试+自动发布
本质：代码改了自动测试并上线
防忽悠：CI/CD=程序员写完代码，系统自动测试、自动部署，不用手动上传
DevOps
 ↔
开发和运维不分家
本质：Development + Operations一体化
防忽悠：DevOps=写代码的和维护服务器的坐一张桌子，出了问题一起扛
第三部分 经济学与金融：术语与缩写还原
GDP (Gross Domestic Product)
 ↔
国内生产总值 / 一年新创造的财富
本质：一个国家一年内生产的最终商品和服务总价值
防忽悠：GDP=国家一年干了多少活，但不等于人民多幸福
GNP (Gross National Product)
 ↔
国民生产总值
本质：一国国民拥有的生产要素生产的总价值


===== 第8页 =====
防忽悠：GNP=中国人无论在哪里赚的钱都算，GDP=在中国地盘上赚的钱
CPI (Consumer Price Index)
 ↔
消费者价格指数 / 物价涨多少
本质：衡量一篮子消费品价格变动的指标
防忽悠：CPI=物价涨幅，超过3%央行就要紧张
PPI (Producer Price Index)
 ↔
工业品出厂价格 / 工厂端通胀
本质：工业企业产品第一次出售时的价格变动
防忽悠：PPI=工厂的进货/出货价格涨了多少，通常比CPI先涨
PMI (Purchasing Managers' Index)
 ↔
采购经理人指数 / 经济景气度
本质：制造业和服务业的景气度指标
防忽悠：PMI=50是分界线，>50扩张，<50收缩
QE (Quantitative Easing)
 ↔
量化宽松 / 央行印钱买债
本质：央行购买长期资产注入流动性
防忽悠：QE=央行直接下场买国债/企业债，印钱给市场，稀释持币者购买力
LPR (Loan Prime Rate)
 ↔
贷款市场报价利率 / 房贷基准
本质：银行对最优质客户的贷款利率
防忽悠：LPR=房贷利率的锚，每月20号公布，房贷合同都参考它
MLF (Medium-term Lending Facility)
 ↔
中期借贷便利 / 央行中期放水
本质：央行向银行提供中期资金的工具
防忽悠：MLF=央行中期放水，期限3个月到1年，利率影响LPR
SLF (Standing Lending Facility)
 ↔
常备借贷便利 / 央行紧急放水
本质：央行向银行提供紧急短期资金的工具
防忽悠：SLF=央行紧急放水，银行缺钱时可以找央行借，俗称酸辣粉
PSL (Pledged Supplementary
Lending)
 ↔
抵押补充贷款 / 央行定向长期放水
本质：央行向政策性银行提供的长期资金
防忽悠：PSL=央行定向长期放水，常用于棚改、基建，俗称胖死了
机会成本 (Opportunity Cost)
 ↔
放弃的最佳选择
本质：为了做A而必须放弃的B的价值


===== 第9页 =====
防忽悠：机会成本=鱼和熊掌不可兼得的量化，你花时间刷抖音，机会成本就是本来能赚的钱
边际效用 (Marginal Utility)
 ↔
再多吃一口的满足感
本质：每多消费一单位商品带来的额外满足
防忽悠：边际效用=吃第七个包子的满足感，通常越吃越不香（递减）
沉没成本 (Sunk Cost)
 ↔
泼出去的水
本质：已经发生且不可收回的成本
防忽悠：沉没成本=电影票买了不好看但钱已经花了，理性人不该考虑但人总是舍不得
纳什均衡 (Nash Equilibrium)
 ↔
谁改谁吃亏的僵持
本质：博弈中没有任何一方能通过单方面改变策略获益
防忽悠：纳什均衡=僵持，如价格战打到双方都没利润，谁先涨价谁丢市场
信息不对称 (Asymmetric
Information)
 ↔
我知道的比你多 / 蒙人
本质：交易双方掌握的信息不同
防忽悠：信息不对称=卖车的知道车况，买车的不知道，信息多的一方有优势
逆向选择 (Adverse Selection)
 ↔
坏人驱逐好人 / 劣币驱逐良币
本质：信息少的一方反而退出市场
防忽悠：逆向选择=二手车市场：好车车主不愿低价卖，市场上只剩烂车
道德风险 (Moral Hazard)
 ↔
有恃无恐
本质：交易后一方因不受监督而采取不利行为
防忽悠：道德风险=买了保险就不锁车，因为有保险公司兜底
外部性 (Externality)
 ↔
副作用
本质：经济活动对第三方产生的影响
防忽悠：外部性=副作用，正外部性=做好事别人沾光，负外部性=做坏事别人遭殃
乘数效应 (Multiplier Effect)
 ↔
一笔钱花出去转几圈
本质：初始支出引起连锁反应导致总收入更大增加
防忽悠：乘数=政府花100万修路，工人拿工资消费…最终GDP增加可能超过100万
不可能三角 (Impossible Trinity)
 ↔
三个好事不能同时占
本质：汇率稳定、资本自由流动、货币政策独立不可兼得


===== 第10页 =====
防忽悠：不可能三角=固定汇率+资本自由+独立货币，三选二
人口红利 (Demographic Dividend)
 ↔
劳动力多且便宜
本质：劳动年龄人口占比高带来的经济增长优势
防忽悠：人口红利=年轻人多、工资低、干活猛，中国过去40年的秘密武器
中等收入陷阱 (Middle-income Trap)
 ↔
发展到中间卡住
本质：人均GDP达到中等水平后增长停滞
防忽悠：中等收入陷阱=从穷到富容易（模仿），从中等到富难（创新），拉美典型
ESG (Environmental, Social,
Governance)
 ↔
环保、社会责任、公司治理
本质：投资评估企业可持续发展的标准
防忽悠：ESG=投资新标准，不看只看利润，还要看是否环保、对员工好、治理规范
VaR (Value at Risk)
 ↔
最坏情况亏多少
本质：给定置信水平下的最大预期损失
防忽悠：VaR=95%概率下明天亏不超过100万，但5%可能亏更多
KPI (Key Performance Indicator)
 ↔
考核指标
本质：衡量工作绩效的关键指标
防忽悠：KPI=你今年必须完成的数字，如销售额1000万
OKR (Objectives and Key Results)
 ↔
目标+关键结果
本质：比KPI更灵活的目标管理方法
防忽悠：OKR=定一个大目标（O），再拆几个关键结果（KR），如O=成为行业第一，KR=用户增长50
%
ROI (Return on Investment)
 ↔
投入产出比
本质：投资收益与成本的比率
防忽悠：ROI=花1块赚几块，ROI>1才值得做
GMV (Gross Merchandise Volume)
 ↔
成交总额（含水分）
本质：一定时间段内的成交总金额
防忽悠：GMV=含退款的成交总额，水分大，实际收入要看净GMV
DAU (Daily Active Users)
 ↔
日活
本质：今天多少人打开APP


===== 第11页 =====
防忽悠：DAU=日活，MAU=月活，两者比值叫用户粘性
LTV (Lifetime Value)
 ↔
一个用户一辈子值多少钱
本质：用户在整个生命周期内贡献的总收入
防忽悠：LTV=用户价值，CAC=拉新成本，LTV>CAC生意才成立
CAC (Customer Acquisition Cost)
 ↔
拉一个新用户花多少钱
本质：获取单个客户的平均成本
防忽悠：CAC=获客成本，投广告花了100万拉了1万人，CAC=100元

===== 第14页 =====
后记
本指南的核心目的不是否定术语的价值——专业交流需要精确——而是建立术语免疫力：当你听到一个新词或
一串字母，能本能地追问"它在描述什么日常现象？""它的数学/物理/生物本质是什么？""谁在通过这个名字
获利？"
记住三条防忽悠法则：
1. 追问全称：遇到缩写先问全称，全称往往已经暴露本质；
2. 还原日常：任何术语都能还原为你已经理解的日常交易、家庭账簿、菜市场博弈；
3. 警惕修辞：凡是带"认知""智能""神经""脑""赋能""引擎""红利"等字眼的商业词汇，先打五折再看。
