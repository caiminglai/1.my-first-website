===== 第1页 =====
医学与心理学黑话还原指南
 从症状到系统论的穿透
医学是术语壁垒最高的领域。同一个生理现象，在临床叫一个名字，在病理学叫另一个名字，在药理学又叫第三个名
字，在患者耳朵里变成不治之症。心理学同样——认知失调、习得性无助、阻抗——本质都是日常情绪的学术包装。
本指南用控制论和系统论的视角，把医学和心理学黑话还原为人体稳态与信息处理的本质。
 整理日期：2026年5月


===== 第2页 =====
前言
人体是一个超稳定系统。医学的每一个术语，本质上都在描述这个系统的某种状态：稳态维持、稳态偏离、稳
态崩溃、稳态重建。控制论的核心概念——反馈、稳态、振荡、崩溃、自组织——在医学中都有精确对应。本
指南按系统层级整理：从细胞分子层面，到器官系统层面，到心理认知层面，再到社会行为层面。


===== 第3页 =====
第一章 细胞与分子：微观层面的稳态语言
细胞凋亡 (Apoptosis)
 ↔
程序性细胞死亡 / 细胞自杀
本质：受控的、有序的细胞死亡过程
防忽悠：凋亡=细胞自杀，是程序化的、不引起炎症的，区别于坏死（他杀）
坏死 (Necrosis)
 ↔
细胞他杀 / 意外死亡
本质：不受控的、被动的细胞死亡
防忽悠：坏死=细胞他杀，细胞膜破裂，内容物泄漏，引起炎症
自噬 (Autophagy)
 ↔
细胞自己吃自己 / 回收利用
本质：细胞降解自身受损组分的过程
防忽悠：自噬=细胞大扫除，吃掉自己的垃圾
氧化应激 (Oxidative Stress)
 ↔
自由基太多 / 体内生锈
本质：活性氧产生与抗氧化防御失衡
防忽悠：氧化应激=体内生锈，就像铁暴露在空气中氧化
炎症 (Inflammation)
 ↔
身体的消防队 / 正反馈初期
本质：免疫系统对损伤或感染的反应
防忽悠：炎症=身体派消防队灭火，短期是好现象
慢性炎症 (Chronic Inflammation)
 ↔
长期低度火灾 / 正反馈失控
本质：持续数周以上的低度炎症
防忽悠：慢性炎症=小火不灭，是很多慢病的共同土壤
细胞因子风暴 / 炎症风暴 (Cytokine
Storm)
 ↔
免疫系统暴走 / 正反馈崩溃
本质：免疫系统过度激活释放大量炎症因子
防忽悠：炎症风暴=免疫系统杀红了眼，不分敌我，自己人打自己人，新冠重症死亡主因。本质就是控
制论中的正反馈失控
受体 (Receptor)
 ↔
锁孔 / 信息接收器
本质：细胞表面接受信号分子的蛋白质
防忽悠：受体=锁孔，激素/药物=钥匙
信号通路 (Signaling Pathway)
 ↔
信息传递链 / 多米诺骨牌


===== 第4页 =====
本质：细胞内信息从受体传递到效应器的级联反应
防忽悠：信号通路=多米诺骨牌，第一块倒了后面一连串反应
基因表达 (Gene Expression)
 ↔
基因开工 / 从图纸到产品
本质：基因信息转化为功能性产物（蛋白质）的过程
防忽悠：基因表达=图纸变成产品
表观遗传 (Epigenetics)
 ↔
基因开关 / 不改变图纸的标注
本质：不改变DNA序列但影响基因表达的修饰
防忽悠：表观遗传=在图纸旁边贴便利贴
代谢 (Metabolism)
 ↔
身体的化工厂 / 物质能量交换
本质：细胞内所有化学反应的总和
防忽悠：代谢=身体的化工厂
稳态 / 内环境稳态 (Homeostasis)
 ↔
内环境稳定 / 动态平衡 /
控制论的稳态
本质：机体维持内部环境相对稳定的能力
防忽悠：内环境稳态=控制论中的稳态（Steady
State）。体温37度、血糖正常、血压稳定，都是负反馈调节在维持
负反馈调节 (Negative Feedback)
 ↔
多了就少、少了就多
本质：输出抑制输入，使系统趋向稳定
防忽悠：负反馈=空调恒温：温度高了制冷，温度低了停
正反馈 (Positive Feedback)
 ↔
越演愈烈 / 雪崩
本质：输出增强输入，使系统偏离稳态
防忽悠：正反馈=麦克风对着音箱：声音→音箱→麦克风→更大声→啸叫

电线控制 / 快准狠
本质：通过神经电信号快速调节器官
防忽悠：神经调节=电线控制，快（毫秒级）、准、短促
体液调节 / 内分泌 (Endocrine)
 ↔
化学快递 / 慢但持久

防忽悠：体液调节=化学快递，慢（秒到小时）、范围广、持久
免疫应答 (Immune Response)
 ↔
body's security system /
识别清除异物
本质：免疫系统识别并清除抗原的过程
防忽悠：免疫应答=body's security system
抗原 (Antigen)
 ↔
坏人的特征码 / 被识别的东西
本质：能被免疫系统识别并引发反应的物质
防忽悠：抗原=特征码，不一定是坏的
抗体 (Antibody)
 ↔
精确制导导弹 / 免疫球蛋白
本质：B细胞产生的特异性识别抗原的蛋白质
防忽悠：抗体=精确制导导弹
补体系统 (Complement System)
 ↔
辅助攻击队 / 补刀
本质：血浆中协助抗体清除病原体的蛋白质群
防忽悠：补体=补刀的，抗体标记敌人后补体上来捅刀
凝血 (Coagulation)
 ↔
身体修水管 / 血小板+纤维蛋白
本质：血液由液态变为凝胶状的过程
防忽悠：凝血=身体修水管
纤溶 (Fibrinolysis)
 ↔
拆脚手架 / 疏通
本质：溶解纤维蛋白凝块的过程
防忽悠：纤溶=拆脚手架
血栓 (Thrombus)
 ↔
血管里的水泥块 / 修过头了
本质：血液在血管内异常凝固形成的固体块
防忽悠：血栓=凝血系统失控
栓塞 (Embolism)
 ↔
血栓跑路 / 堵塞远处
本质：血栓脱落随血流堵塞其他血管
防忽悠：栓塞=血栓搬家
动脉粥样硬化 (Atherosclerosis)
 ↔
血管生锈结垢 / 慢性炎症

本质：动脉壁脂质沉积、纤维化、钙化
防忽悠：动脉粥样硬化=水管生锈结垢
斑块 (Plaque)
 ↔
血管壁上的垃圾堆 / 定时炸弹
本质：动脉粥样硬化形成的隆起性病变
防忽悠：斑块=垃圾堆，稳定斑块=暂时没事，不稳定斑块=随时塌方
代偿 (Compensation)
 ↔
器官加班 / 超稳定系统切换
本质：器官功能受损时其他部分或自身增强功能
防忽悠：代偿=超稳定系统的自动切换。一个肾坏了另一个肾加班，心肌肥厚维持泵血。本质就是控制
论中的超稳定系统（Ultrastable System）
失代偿 (Decompensation)
 ↔
加班也干不完 / 超稳定崩溃
本质：代偿机制不足以维持功能，系统崩溃
防忽悠：失代偿=加班也撑不住了，超稳定系统切换失败
多器官功能障碍 (MODS)
 ↔
多米诺骨牌倒塌 / 系统级联崩溃
本质：一个器官衰竭引发其他器官相继衰竭
防忽悠：MODS=多米诺骨牌，系统级联崩溃
第三章 症状与诊断：临床术语的日常翻译
发热 (Fever)
 ↔
体温调定点上移 / 免疫重设恒温器
本质：免疫反应导致下丘脑体温设定点升高
防忽悠：发烧=身体把恒温器调高，为了抑制病原体繁殖
炎症反应 (Inflammatory Response)
 ↔
红肿热痛 / 消防队四件套
本质：局部血管扩张、通透性增加、白细胞渗出
防忽悠：红肿热痛=消防队四件套
休克 (Shock)
 ↔
全身断水断电 / 灌注不足
本质：有效循环血量锐减，组织灌注不足
防忽悠：休克=全身断水断电
缺氧 (Hypoxia)
 ↔
细胞断粮 / 没氧气
本质：组织供氧不足或利用障碍

防忽悠：缺氧=细胞断粮

防忽悠：水肿=下水道堵了

综合征 (Syndrome)

第四章 药理学：药物作用的系统论
激动剂 (Agonist)
 ↔
模拟钥匙开门的药
本质：与受体结合并激活受体产生效应
防忽悠：激动剂=假钥匙也能开门
拮抗剂 (Antagonist)
 ↔
堵锁孔的药 / 占位不干活
本质：与受体结合但不激活，阻断其他物质作用
防忽悠：拮抗剂=占着茅坑不拉屎
半衰期 (Half-life)
 ↔
药效减半的时间
本质：药物浓度下降一半所需时间
防忽悠：半衰期=药效减半的时间
生物利用度 (Bioavailability)
 ↔
吃进去能用的比例
本质：药物进入体循环的比例和速度
防忽悠：生物利用度=吃100mg，实际进入血液的可能只有70mg
首过效应 (First-pass Effect)
 ↔
肝脏先截留 / 过路费
本质：口服药物经肝脏代谢后进入体循环的量减少
防忽悠：首过效应=口服药先过肝脏这关
耐受 (Tolerance)
 ↔
药越吃越没用 / 受体下调
本质：反复用药后药效下降，需加大剂量
防忽悠：耐受=身体适应了
依赖 (Dependence)
 ↔
停药就难受 / 稳态重建


===== 第9页 =====
本质：长期用药后身体适应，停药出现戒断症状
防忽悠：依赖=身体把药物当成正常稳态的一部分
成瘾 (Addiction)
 ↔
大脑奖赏回路劫持 / 多巴胺绑架
本质：强迫性觅药行为，伴随神经适应性改变
防忽悠：成瘾=大脑被多巴胺绑架
副作用 (Side Effect)
 ↔
顺手干了别的事 / 非目标作用
本质：药物在治疗剂量下出现的非预期作用
防忽悠：副作用=药物不只干一件事
毒性 (Toxicity)
 ↔
药吃太多了 / 过犹不及
本质：药物剂量过大或蓄积过多引起的损害
防忽悠：毒性=过犹不及
治疗窗 (Therapeutic Window)
 ↔
安全有效的剂量范围
本质：最低有效浓度与最低中毒浓度之间的范围
防忽悠：治疗窗=安全区
安慰剂效应 (Placebo Effect)
 ↔
相信就好使 / 心理-生理耦合
本质：无药理活性的制剂产生治疗效果
防忽悠：安慰剂=相信就好使
双盲试验 (Double-blind)
 ↔
医生和患者都不知道谁吃了真药
本质：试验双方均不知分组情况
防忽悠：双盲=防止偏见
第五章 心理学：认知与情绪的系统论
认知失调 (Cognitive Dissonance)
 ↔
想法打架 / 信息冲突
本质：同时持有两种矛盾认知产生的心理不适
防忽悠：认知失调=心里两个声音吵架
确认偏误 (Confirmation Bias)
 ↔
只信自己想信的 / 信息滤波
本质：倾向于寻找、解释、记忆支持自己已有信念的信息


===== 第10页 =====
防忽悠：确认偏误=信息滤波，只让符合自己观点的信息通过。本质就是控制论中的选择性滤波
可得性启发 (Availability Heuristic)
 ↔
容易想起的就觉得常见
本质：根据事件在记忆中的易得性判断概率
防忽悠：可得性启发=最近看到空难新闻就觉得飞机危险
锚定效应 (Anchoring Effect)
 ↔
第一印象定调子
本质：过度依赖最初获得的信息做判断
防忽悠：锚定=先报高价再打折你觉得便宜
损失厌恶 (Loss Aversion)
 ↔
亏比赚更难受
本质：同等损失的痛苦大于同等收益的快乐
防忽悠：损失厌恶=亏1000的痛苦 > 赚1000的快乐
禀赋效应 (Endowment Effect)
 ↔
自己的东西更值钱
本质：拥有某物后对其估值上升
防忽悠：禀赋效应=自己的杯子估价更高
心理账户 (Mental Accounting)
 ↔
钱分门别类放不同口袋
本质：对不同来源/用途的钱有不同心理分类
防忽悠：心理账户=工资舍不得花，彩票赢的随便花
沉没成本谬误 (Sunk Cost Fallacy)
 ↔
舍不得已经花的钱
本质：因已投入成本而继续不合理行为
防忽悠：沉没成本谬误=电影票难看但舍不得走
习得性无助 (Learned Helplessness)
 ↔
屡败屡战到不想战了
本质：反复经历不可控负面事件后放弃努力
防忽悠：习得性无助=电击狗无法逃避，后来能逃也不逃了
自我效能感 (Self-efficacy)
 ↔
我能行的感觉 / 掌控感
本质：对自己完成某项任务能力的信念
防忽悠：自我效能=我能行的感觉
马斯洛需求层次 (Maslow's
Hierarchy)
 ↔
需求金字塔 / 从生存到自我实现
本质：生理→安全→社交→尊重→自我实现

防忽悠：马斯洛=需求升级打怪
心流 (Flow)
 ↔
全神贯注忘我状态

 ↔

本质：经历创伤后持续的过度警觉和闪回

本质：有意识地关注当下经验
防忽悠：正念=专注当下

改想法改行为 / 系统调试
本质：通过改变认知和行为来改善情绪
防忽悠：CBT=系统调试

后记
医学和心理学不是两个孤立的领域，它们共享同一套系统论语言。人体是一个超稳定系统，心理是信息处理系
统。疾病=稳态偏离或崩溃，治疗=重建稳态，康复=系统自适应。当你理解炎症风暴就是控制论的正反馈崩溃
，代偿机制就是超稳定系统的自动切换，内环境稳态就是控制论的稳态，心理防御机制就是信息滤波——你就
不再是被术语吓住的患者或家属，而是能穿透表象看到系统本质的观察者。
