计算机与AI概念防忽悠指南
 
 AI时代黑话还原手册
 
AI领域每天都在造新词。大模型、涌现、对齐、RLHF、RAG、思维链……这些听起来高大上的术语，本质几乎都是传
统计算机科学、数学、统计学、心理学里已有概念的重新包装。看懂包装纸，就能不被忽悠。
 整理日期：2026年5月

前言
AI行业有一个特点：老酒换新瓶的速度极快。同一个算法思想，今年叫深度学习，明年叫大模型，后年叫Agen
t；同一个数学操作，在论文里叫注意力机制，在工程里叫加权求和，在产品发布会上叫认知引擎。本手册把A
I黑话还原为传统CS/数学概念，每一组都给出：AI黑话 ↔ 传统概念，本质一句话，以及防忽悠提示。

第一章 神经网络本质：函数拟合
神经网络
↔
复杂函数拟合器 / 万能逼近器
本质：多层非线性函数复合，用数据拟合任意映射
防忽悠：神经网络听起来像大脑，本质就是一个有很多参数的函数
前向传播
↔
函数求值
本质：从输入到输出逐层计算
防忽悠：前向传播=代入函数算结果
反向传播
↔
链式法则求导 (微积分)
本质：从输出误差反向计算每层参数的梯度
防忽悠：反向传播=复合函数求导的链式法则，高中导数知识
梯度下降
↔
沿着最陡方向下山 (优化)
本质：按损失函数的负梯度方向更新参数
防忽悠：梯度下降=看哪边最陡就往反方向走一步
学习率
↔
步长
本质：每次更新参数走多远
防忽悠：学习率=步长，太大容易摔，太小走得慢
损失函数
↔
误差度量 / 目标函数 / 代价函数
本质：衡量模型输出与真实值差距的函数
防忽悠：损失=误差=代价=目标，同一个东西四个名字
正则化
↔
惩罚项 / 先验约束 / 不要想太多
本质：在损失函数里加惩罚防止参数过大
防忽悠：正则化=给模型戴紧箍咒
过拟合
↔
死记硬背 / 高方差 / 学偏了
本质：模型记住了训练数据的噪声，对新数据表现差
防忽悠：过拟合=死记硬背=只会背题不会变通
欠拟合
↔
没学懂 / 高偏差
本质：模型太简单，连训练数据规律都没抓住


--- Page 4 ---
防忽悠：欠拟合=没学懂=教了也不会
泛化
↔
举一反三 / 对新数据也管用
本质：模型在未见过的数据上表现好
防忽悠：泛化能力强=举一反三
激活函数
↔
非线性门 / 决定传不传
本质：引入非线性，决定神经元输出什么
防忽悠：激活函数=开关/阀门
ReLU
↔
负数归零，正数不变 / 整流
本质：max(0,x)
防忽悠：ReLU=线性整流，最简单的非线性
Sigmoid
↔
S型曲线 / 压缩到0-1
本质：1/(1+e^(-x))
防忽悠：Sigmoid=S型函数，把任意数压到0和1之间
Softmax
↔
归一化指数 / 多分类概率输出
本质：把一组数变成概率分布
防忽悠：Softmax=先取指数再归一化
Dropout
↔
随机失活 / 临时砍掉神经元
本质：训练时随机忽略部分神经元，防止过拟合
防忽悠：Dropout=随机临时裁员
批归一化
↔
统一批次尺度 / 稳定数据分布
本质：对一个批次的数据做标准化
防忽悠：批归一化=把一批数据的均值变0、方差变1
层归一化
↔
按样本统一尺度
本质：对每个样本的所有特征做标准化
防忽悠：层归一化=对单个样本内部做标准化
残差连接 / Skip Connection
↔
抄近路 / 梯度高速公路
本质：把输入直接加到输出上
防忽悠：残差=抄近路，让梯度直接从深层传回浅层

梯度消失
↔
信号越传越弱
本质：反向传播时梯度逐层衰减到接近0
防忽悠：梯度消失=信号衰减
梯度爆炸
↔
信号越传越强
本质：反向传播时梯度逐层放大到无穷大
防忽悠：梯度爆炸=信号放大失控
早停
↔
见好就收
本质：验证集性能不再提升就停止训练
防忽悠：早停=见好就收
检查点
↔
存档点
本质：保存模型参数的某个状态
防忽悠：检查点=游戏存档
批次 / Batch
↔
一小撮数据
本质：每次更新参数用的样本子集
防忽悠：批次=一小撮
轮次 / Epoch
↔
全部数据过一遍
本质：整个训练集被遍历一次
防忽悠：一轮=全部数据看一遍
迭代 / Iteration
↔
一次参数更新
本质：每完成一个批次的训练并更新一次参数
防忽悠：一次迭代=看一批数据、算梯度、改一次参数
超参数
↔
人工设定的配置项
本质：模型结构和学习过程的预设值
防忽悠：超参数=人定的配置
网格搜索
↔
穷举试错
本质：把所有超参数组合试一遍
防忽悠：网格搜索=穷举法
随机搜索
↔
随机试错


--- Page 6 ---
本质：随机采样超参数组合
防忽悠：随机搜索=瞎蒙
贝叶斯优化
↔
智能试错
本质：根据已有结果预测下一个可能好的点
防忽悠：贝叶斯优化=基于经验聪明地猜
第二章 深度学习架构：旧瓶装新酒
卷积 / Convolution
↔
滑动窗口加权求和 / 滤波器
本质：用一个核函数在输入上滑动做加权平均
防忽悠：卷积=滑动窗口滤波，图像处理里叫滤波
池化 / Pooling
↔
降采样 / 取区域代表值
本质：把相邻区域压缩成一个数
防忽悠：池化=降采样，最大池化=取区域最大值
卷积神经网络 / CNN
↔
带滤波器的神经网络
本质：用卷积层提取局部特征的神经网络
防忽悠：CNN=用滑动窗口做特征提取的网络
循环神经网络 / RNN
↔
带记忆的神经网络
本质：处理序列数据，隐含状态携带历史信息
防忽悠：RNN=有短期记忆的网络
长短期记忆 / LSTM
↔
改进的RNN / 长期记忆单元
本质：用门控机制解决RNN长期依赖问题
防忽悠：LSTM=给RNN加了遗忘门和记忆门
门控循环单元 / GRU
↔
简化的LSTM
本质：用更少的门实现类似LSTM的效果
防忽悠：GRU=LSTM的精简版
Transformer
↔
带加权求和的深度网络 /
抛弃RNN的序列模型
本质：完全基于注意力机制，并行处理序列


--- Page 7 ---
防忽悠：Transformer=不用循环、全靠注意力加权求和的网络
注意力机制 / Attention
↔
加权求和 (小学平均数升级版)
本质：根据相关性给不同位置分配权重
防忽悠：注意力=加权求和，小学算平均分也是加权求和
自注意力 / Self-Attention
↔
自己跟自己算相关性
本质：序列内部元素互相计算关联权重
防忽悠：自注意力=句子里的词互相看
多头注意力 / Multi-Head
↔
多组自注意力并行 / 多视角观察
本质：用多组不同的权重矩阵并行计算注意力
防忽悠：多头=多视角
位置编码 / Positional Encoding
↔
给序列加序号信息
本质：让模型知道每个词在句子中的位置
防忽悠：位置编码=给每个位置贴个坐标标签
前馈网络 / FFN
↔
全连接层 / 每个点都连
本质：注意力之后的非线性变换层
防忽悠：前馈层=普通全连接神经网络
编码器 / Encoder
↔
压缩输入成表示
本质：把输入序列映射为内部表示
防忽悠：编码器=压缩/提炼
解码器 / Decoder
↔
从表示生成输出
本质：把内部表示还原为输出序列
防忽悠：解码器=解压/生成
掩码 / Mask
↔
遮住不让看
本质：隐藏部分输入，防止偷看答案
防忽悠：掩码=盖住
掩码语言模型
↔
完形填空
本质：遮住部分词让模型预测
防忽悠：掩码语言模型=完形填空

生成式预训练 / GPT
↔
自回归语言模型 /
从左到右猜下一个词
本质：用前文预测下一个词
防忽悠：GPT=给定前面所有词，预测下一个词
双向编码器 / BERT
↔
双向完形填空
本质：同时看左右上下文预测被遮住的词
防忽悠：BERT=双向完形填空
图神经网络 / GNN
↔
在图上做神经网络 / 邻居信息聚合
本质：把节点及其邻居的信息聚合更新
防忽悠：GNN=让每个节点跟邻居聊天，更新自己的状态
第三章 大模型与AI应用：新词轰炸区
大模型 / LLM
↔
超级自动补全 / 下一个词预测器
本质：参数量巨大的语言模型，基于概率预测下一个token
防忽悠：大模型=参数量巨大的自动补全引擎
涌现能力 / Emergence
↔
量变引起质变 / 规模大到突然会了
本质：模型规模超过阈值后突然表现出新能力
防忽悠：涌现=量变引起质变
提示工程 / Prompt Engineering
↔
出题措辞优化
本质：设计输入文本的表达方式以获得更好输出
防忽悠：提示工程=怎么问问题
对齐 / Alignment
↔
价值观调教 / 输出符合人类偏好
本质：让模型行为符合人类意图和伦理
防忽悠：对齐=调教
RLHF
↔
人工打分的强化学习
本质：用人类反馈训练奖励模型，再用强化学习优化策略
防忽悠：RLHF=让人给模型打分，模型根据分数调整自己
嵌入 / Embedding
↔
编码成坐标 / 向量表示


--- Page 9 ---
本质：把离散对象映射到连续向量空间
防忽悠：嵌入=给每个东西发一个坐标
向量数据库
↔
最近邻搜索 / 空间索引
本质：存储向量并快速找相似向量
防忽悠：向量数据库=专门找相近坐标的数据库
RAG
↔
开卷考试 / 检索+生成
本质：先查资料再回答，外挂知识库
防忽悠：RAG=开卷考试
微调 / Fine-tuning
↔
专项训练 / 在预训练基础上做迁移
本质：在大模型基础上用特定领域数据继续训练
防忽悠：微调=学会通用知识后，再学专业课程
零样本学习 / Zero-shot
↔
没学过也会做
本质：模型在没有见过某类任务示例的情况下直接执行
防忽悠：零样本=没举例子直接考
少样本学习 / Few-shot
↔
举一反一 / 举三反一
本质：给几个例子就能学会新任务
防忽悠：少样本=举几个例子就懂
思维链 / CoT
↔
把思考过程写出来 / 显式推理
本质：让模型逐步展示推理步骤
防忽悠：思维链=要求模型写草稿纸
多模态 / Multimodal
↔
同时处理图文音视频 /
跨感官信息融合
本质：模型能理解和生成多种类型的数据
防忽悠：多模态=眼睛+耳朵+嘴一起用
扩散模型 / Diffusion
↔
从噪声中逐步修图 / 随机过程逆推
本质：通过逐步去噪生成数据
防忽悠：扩散模型=从一张纯雪花屏开始，一点点修成清晰图片
GAN
↔
造假者与鉴定师的博弈 /
Minimax博弈


--- Page 10 ---
本质：生成器和判别器互相训练
防忽悠：GAN=造假者和鉴定师打架
VAE
↔
压缩+重建 / 有损编码
本质：学习数据的压缩表示再重建
防忽悠：VAE=先压缩成密码，再从密码还原
强化学习 / RL
↔
试错学习 / 奖惩训练
本质：通过奖励和惩罚学习最优策略
防忽悠：强化学习=训狗
策略 / Policy
↔
行为规则 / 映射：状态→动作
本质：给定状态下选择动作的规则
防忽悠：策略=遇到什么情况就怎么办
价值函数 / Value Function
↔
未来总回报的期望 / 评分函数
本质：评估某个状态或动作有多好
防忽悠：价值=从长远看这一步能赚多少分
Q学习 / Q-Learning
↔
查表选动作 / 动作价值迭代
本质：学习状态-动作对的长期价值
防忽悠：Q=Quality of action，每个状态下每个动作打个分
马尔可夫决策过程 / MDP
↔
无记忆随机决策模型
本质：未来只依赖当前状态，与历史无关
防忽悠：马尔可夫=无记忆
智能体 / Agent
↔
做决策的实体 / 机器人/程序/玩家
本质：在环境中感知和行动的主体
防忽悠：智能体=能感知、能决策、能行动的任何东西
环境 / Environment
↔
外部世界
本质：智能体所处的外部系统
防忽悠：环境=智能体以外的所有东西
状态 / State
↔
当前情境的完整描述
本质：环境在某一时刻的快照


--- Page 11 ---
防忽悠：状态=现在什么情况
动作 / Action
↔
选择
本质：智能体在某状态下可以采取的行为
防忽悠：动作=我能干什么
奖励 / Reward
↔
即时反馈分数
本质：环境对智能体动作的即时评价
防忽悠：奖励=即时打分
第四章 机器学习传统概念：名字迷惑区
监督学习
↔
有答案的题海战术
本质：用带标签的数据训练
防忽悠：监督学习=老师批改作业
无监督学习
↔
自己找规律
本质：用无标签的数据发现结构
防忽悠：无监督学习=自己摸索
半监督学习
↔
大部分没答案，少部分有答案
本质：少量标签+大量无标签数据联合训练
防忽悠：半监督=10道题里1道有答案
自监督学习
↔
自己出题自己做
本质：从数据本身构造监督信号
防忽悠：自监督=遮住图片一半猜另一半
对比学习
↔
找不同/找相同
本质：拉近同类样本、推开异类样本
防忽悠：对比学习=正样本拉近、负样本推开
生成式AI
↔
自动创作机
本质：生成新内容（文本、图像、音频）
防忽悠：生成式=能写能画能作曲

判别式AI
↔
分类/判断/识别
本质：区分不同类别
防忽悠：判别式=是猫还是狗
表征学习
↔
自动提炼特征 / 端到端特征工程
本质：让模型自己学习有用的数据表示
防忽悠：表征学习=自动找规律
特征工程
↔
人工设计输入变量
本质：根据领域知识构造模型输入
防忽悠：特征工程=人脑找规律喂给模型
端到端 / End-to-End
↔
直接输入输出，中间不人工干预
本质：从原始输入直接到最终输出
防忽悠：端到端=黑箱直达
黑箱模型
↔
不知道内部怎么算的
本质：输入输出可观测，内部机制不可解释
防忽悠：黑箱=只知道输入输出
可解释性
↔
说清楚为什么这样判断
本质：让模型决策过程对人类透明
防忽悠：可解释性=给黑箱开个小窗
对抗样本
↔
特意设计的欺骗输入 / 视觉错觉
本质：微小扰动导致模型误判
防忽悠：对抗样本=给图片加人眼看不出的噪声
鲁棒性
↔
抗干扰能力 / 稳定性
本质：模型在噪声、攻击、分布变化下保持性能
防忽悠：鲁棒=皮实
迁移学习
↔
举一反三 /
学会骑自行车再骑摩托车
本质：在一个任务上学到的知识用到另一个任务
防忽悠：迁移=举一反三

域适应
↔
换个环境还能用
本质：模型从源领域适应到目标领域
防忽悠：域适应=在中国学的驾驶技术，到美国也能用
元学习
↔
学会如何学习 / 学习的学习
本质：学习一个能快速适应新任务的初始化或策略
防忽悠：元学习=授人以渔
联邦学习

数据不出本地的协作学习
本质：多个参与方在不共享原始数据的情况下联合训练
防忽悠：联邦学习=各医院数据不出门，只传模型参数
知识蒸馏
↔
大模型教小模型 / 传功
本质：用大模型的输出训练小模型
防忽悠：蒸馏=老师教学生
模型剪枝

后记
AI领域的术语爆炸速度远超其他学科。同一个思想，在学术界叫一个名字，在开源社区叫另一个名字，在创业
公司PR稿里又换一个名字。本手册的目的不是穷尽所有黑话，而是建立概念抗体：当你听到一个新词，能本
能地追问它的数学本质是什么、它和五年前哪个概念是同一回事。
