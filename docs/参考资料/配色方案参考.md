## 1. Hex 转 HSL 转换说明

### Tailwind CSS 变量格式

Tailwind CSS v3+ 使用 CSS 变量定义颜色时，格式为 **不带 `hsl()` 包裹**的纯数值：

```
/* 格式: H S% L% (不带 hsl() 外壳) */
--primary: 221 83% 53%;

/* 在 Tailwind 中使用 */
@apply bg-primary;  /* 自动包裹为 hsl(var(--primary)) */
```

### 转换公式

```
Hex #RRGGBB -> RGB (0-255) -> 归一化 (0-1) -> HSL

H (色相): 0-360 (在 Tailwind 中只写数字)
S (饱和度): 0%-100%
L (亮度): 0%-100%
```

### 在 tailwind.config.js 中引用

```js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: 'hsl(var(--primary) / <alpha-value>)',
        // 展开为: hsl(221 83% 53% / var(--tw-text-opacity))
      }
    }
  }
}
```

## 3. UI 风格速查表

以下是从 styles.csv 中提取的核心 UI 风格及其关键特征：

### 3.1 Glassmorphism (玻璃态)

| 属性 | 值 |
|------|-----|
| **关键词** | glass, frosted, blur, translucent, depth, premium |
| **核心特征** | 半透明背景 + 高斯模糊 + 细微边框 + 渐变光影 |
| **主色** | Primary Violet `#7C3AED`, Cyan `#06B6D4` |
| **背景** | 深色渐变 `#0F172A` -> `#1E293B` |
| **适用** | 高端品牌、金融科技、音乐/流媒体、仪表盘 |
| **注意** | 性能消耗较大(模糊效果)，需 GPU 支持 |

### 3.2 Neumorphism (新拟态)

| 属性 | 值 |
|------|-----|
| **关键词** | soft UI, dual shadow, extruded, inset, ceramic |
| **核心特征** | 同色系背景 + 双重阴影(亮/暗) + 凹凸效果 |
| **主色** | Accent `#6C63FF`, Base `#E0E5EC` |
| **背景** | 单色 `#E0E5EC` (Cool Clay) |
| **适用** | 智能家居控制、极简工具、健康监控 |
| **注意** | 对比度低，需仔细调整 WCAG 合规性 |

### 3.3 Brutalism (粗野主义)

| 属性 | 值 |
|------|-----|
| **关键词** | raw, bold, monospace, high-contrast, stark |
| **核心特征** | 系统字体 + 高对比度 + 无装饰 + 粗边框 |
| **主色** | Pure Black `#000000`, White `#FFFFFF` |
| **适用** | 开发者作品集、实验性设计、技术艺术 |
| **注意** | 不适合需要温馨感的品牌 |

### 3.4 Neo Brutalism (新粗野主义)

| 属性 | 值 |
|------|-----|
| **关键词** | pop art, stickers, thick borders, cream, offset shadow |
| **核心特征** | 4px 粗黑边框 + 硬偏移阴影(无模糊) + 波普色彩 + 微旋转 |
| **主色** | Cream `#FFFDF5`, Hot Red `#FF6B6B`, Yellow `#FFD93D` |
| **字体** | Space Grotesk Bold/Black only (700-900) |
| **适用** | Gen Z 营销、创意工具、贴纸风格应用 |
| **注意** | 不适合严肃企业或保守行业 |
