# 模拟IC设计工程师

**行业黑话翻译：**
设计运放、LDO、ADC这些模拟电路芯片。模拟IC是芯片界的"老中医"——越老越值钱，因为经验无法被AI替代。

**实际工作内容（人话版）：**
设计运放、LDO、ADC这些模拟电路芯片。模拟IC是芯片界的"老中医"——越老越值钱，因为经验无法被AI替代。

**薪资真相：**
15K-30K

**入门门槛：**
学历：本科/硕士
经验：6-12个月入门

**学习阶段：**
| 阶段 | 学什么 | 产出 |
|------|--------|------|
| Stage 1：筑基期（1-3个月） | MOS管工作原理、单级放大器、电流镜与偏置、Cadence Virtuoso安装与基本操作 | |
| Stage 2：实战期（3-6个月） | 差分对设计、频率补偿与稳定性、LDO完整设计、版图设计基础 | |
| Stage 3：进阶期（6-12个月） | Bandgap基准源、噪声分析、ADC/DAC架构、Spec解读与系统权衡 | |

**必备技能：**
- MOS管工作原理
- 单级放大器
- 电流镜与偏置
- Cadence Virtuoso安装与基本操作
